Descrição sobre C++:

É uma linguagem de programação compilada multi-paradigma (seu suporte inclui linguagem imperativa, orientada a objetos e genérica) e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores. Bjarne Stroustrup desenvolveu o C++ (originalmente com o nome C with Classes, que significa C com classes em português) em 1983 no Bell Labs como um adicional à linguagem C. Novas características foram adicionadas com o tempo[5], como funções virtuais, sobrecarga de operadores, herança múltipla, gabaritos e tratamento de exceções. Após a padronização ISO realizada em 1998 e a posterior revisão realizada em 2003, uma nova versão da especificação da linguagem foi lançada em dezembro de 2014, conhecida informalmente como C++17.

Vantagens e Desvantagens de utilizar C++

Vantagens Desvantagens
Fácil de usar do que outras linguagens de baixo nível Não oferece suporte para coleta de lixo
Tem muitas vantagens quando envolve programação Programas C++ são frequentemente pesados
Usa programação multiparadigma C++ é complexo durante um programa de alto nível muito grande

Vantagens:

Como a codificação binária, requer muito mais espaço do que as linguagens de baixo nível, mas é muito mais fácil de entender e aprender.

Todos os arquivos autônomos do programa C++ devem usar a função principal para permitir que o programa inicie e motive suas funções.

Planejamento da programação, paradigma preocupado com a lógica, a estrutura e o procedimento do programa, o programa C++ é multiparadigma significa que segue três paradigmas Genérico, Imperativo e Orientado a Objetos.


Desvantagens:

O programa C++ não oferece suporte para coleta de lixo, não oferece suporte para alocação de memória dinâmica

Os programas C++ são frequentemente pesados e se não for cuidadoso, o programa C++ permite classes e, portanto, as funções com um nome equivalente e funções sobrecarregadas, portanto, o sistema de mutilação de símbolos deve ser usado. No entanto, ele pode ser facilmente envolvido em funções C.

C++ é comumente empregado para aplicativos específicos da plataforma. Para o sistema operacional ou plataforma real, o conjunto de bibliotecas é normalmente escolhido.
